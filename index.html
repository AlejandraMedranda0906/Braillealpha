Objetivos

Objetivo General

Desarrollar un dispositivo electrónico para el proceso de aprendizaje del sistema Braille

1

Identificar las tecnologías adecuadas para la creación del dispositivo electrónico Braille.

2

Diseñar un prototipo funcional que permita al usuario aprender el alfabeto Braille mediante interacción táctil y retroalimentación auditiva.

3

Registrar en una base de datos en la nube las letras logradas por el usuario, para identificar su progreso cuando sea necesario.

Conclusión

Se logró identificar la necesidad de usar el microcontrolador ESP32 como una solución viable para la realización del dispositivo. Asimismo, para el diseño del dispositivo se vio necesario utilizar herramientas tecnológicas como Arduino IDE para su respectiva codificación y funcionalidad, y el registro de datos en una base de datos en la nube de manera manual. Se proyecta realizar una conexión a la nube de manera sistemática como posible mejora para el dispositivo. Como conclusión, se considera al dispositivo electrónico denominado AlphaBraille como un buen dispositivo para iniciar de manera divertida el aprendizaje del abecedario en Braille.
